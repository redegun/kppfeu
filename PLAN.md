## 🛠 Стек технологий

| Компонент | Технология | Почему |
|-----------|-----------|--------|
| Фреймворк | **Astro 5** | SSG, быстрый, SEO-friendly, минимум ресурсов на 1GB VPS |
| UI-компоненты | **Preact** (island architecture) | Лёгкий React-совместимый для интерактивных элементов (форма, корзина) |
| Стили | **Tailwind CSS 4** | Быстрая верстка, утилитарный подход, маленький bundle |
| Формы/заказы | **Astro API routes + Nodemailer** | Отправка заявок на email без внешних сервисов |
| Веб-сервер | **Nginx** (reverse proxy) + **Node.js** (Astro SSR) | Либо полный SSG + Nginx static |
| Деплой | **GitHub Actions → VPS** | Автодеплой по push в main |
| SSL | **Let's Encrypt (Certbot)** | Бесплатный HTTPS |

### Альтернатива: Full Static (SSG)
Если форма заказа реализуется через внешний сервис (Formspree/Getform), можно обойтись чистым SSG + Nginx без Node.js runtime. Это ещё легче для 1GB VPS.

## 🛒 Логика заказа

1. Пользователь выбирает товар → добавляет в корзину (кол-во рулонов)
2. В корзине → форма заказа:
   - Имя
   - Email
   - Телефон
   - Товары (автозаполнение из корзины)
3. Отправка → email менеджеру
4. Без оплаты онлайн (запрос → менеджер связывается)
